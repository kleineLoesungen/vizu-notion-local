Visualize Your Notion Workspace

Render your Notion data as metro maps, process flows, or fully custom Mermaid diagrams — no code changes required.

View on GitHub Docker Hub

Features

⚙

Config-Driven

Define Notion sources and column mappings in a JSON config file. Update the config, restart the container, and the app validates and reloads — no code changes.

◉

Built-in Visualization Types

Hierarchical databases render as metro-style maps. Sequential/workflow databases render as process flows. The app auto-detects which type fits each source.

▶

Fully Local & Self-Hosted

Single docker-compose up command. Your Notion token stays server-side. No cloud relay, no external dependencies — run on your own machine.

◈

Custom Diagram Templates

Drop a .mmd template file into the config folder and restart — any Mermaid diagram type rendered with live Notion data. No code changes required.

How It Works

01
Configure

Edit a simple JSON config file with your Notion database IDs and column mappings.

02
Run

Start the container with docker-compose up. The app validates your config and connects to Notion.

03
Visualize

Open the browser and view your data as metro maps, process flows, or custom Mermaid diagrams. Filter, export SVG, and share links. For custom diagram types, drop a .mmd Handlebars template into the config folder and restart.

Built for Local Use

This tool runs on your machine for your eyes. Scope is intentionally narrow.

Not included

User Management

No accounts, no login, no sessions. Anyone who can reach port 3000 on your machine can use the app.

Not included

Access Control

No roles, no read/write separation, no API keys for the UI. Secure it at the network level if needed.

Not included

Backup

No data is written to disk except share link tokens in ./data. Your Notion workspace is the source of truth.

Not included

Health Checks & Alerting

No uptime monitoring, no alerting, no SLA. If the container exits, restart it with docker compose up.

Built with Nuxt + Vue 3 + TailwindCSS. Config-driven visualization for Notion. Metro map rendering powered by Metroviz. Custom diagrams powered by Mermaid.
