Add collapse state initialized to `true` (collapsed) — override to `false` on wide screens:
```typescript
const isOpen = ref(true)  // default collapsed; overridden to open on wide screens

onMounted(() => {
  // Expand by default on screens wider than 768px (md breakpoint)
  if (window.matchMedia('(min-width: 768px)').matches) {

})
```
