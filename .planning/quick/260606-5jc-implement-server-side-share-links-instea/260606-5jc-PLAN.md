**server/routes/api/share.post.ts**: POST endpoint that:
- Reads body as `ViewState` JSON via `readBody(event)`
- Generates an 8-char alphanumeric token: `Math.random().toString(36).slice(2, 10)` (no external lib)
- Writes the JSON to `/app/data/shares/{token}.json` using Node `fs/promises`: `mkdir -p /app/data/shares` then `writeFile`
- Returns `{ token }` as JSON

Full implementation:
```typescript
import { mkdir, writeFile } from 'node:fs/promises'
import { join } from 'node:path'

export default defineEventHandler(async (event) => {
  const body = await readBody(event)
  const token = Math.random().toString(36).slice(2, 10)
  const dir = join('/app/data/shares')
  await mkdir(dir, { recursive: true })
  await writeFile(join(dir, `${token}.json`), JSON.stringify(body))
  return { token }
})
```

**server/routes/api/share/[token].get.ts**: GET endpoint that:
- Reads `token` from `event.context.params`
- Reads `/app/data/shares/{token}.json` with `readFile`
- Returns parsed JSON as-is
- On ENOENT: throws 404 via `createError({ statusCode: 404, message: 'Share not found' })`

Full implementation:
```typescript
import { readFile } from 'node:fs/promises'
import { join } from 'node:path'

export default defineEventHandler(async (event) => {
  const { token } = event.context.params as { token: string }
  try {
    const raw = await readFile(join('/app/data/shares', `${token}.json`), 'utf-8')
    return JSON.parse(raw)
  } catch (err: any) {
    if (err.code === 'ENOENT') {
      throw createError({ statusCode: 404, statusMessage: 'Not Found', message: 'Share not found' })
    }
    throw err
  }
})
```

Note: The Dockerfile runs as non-root user `nuxt` (uid 1001). When `docker-compose up` is run on a fresh checkout, Docker creates the `./data` host directory owned by root. To ensure writability, the user must `mkdir -p ./data/shares && chmod 777 ./data/shares` once on the host — document this in the docker-compose comment or in a README note. Alternatively, add an `entrypoint` script that chowns the dir; but for simplicity just use `chmod 777` on the host dir and note it.

Actually simpler: in docker-compose.yml add a note comment. The `mkdir` with `recursive: true` in the POST handler handles the case where the dir doesn't exist inside the container if the volume is mounted with correct perms.

For local dev (non-Docker), the path `/app/data/shares` won't exist. Add a `DATA_DIR` env fallback:
```typescript
const sharesDir = process.env.DATA_DIR
  ? join(process.env.DATA_DIR, 'shares')
  : join('/app/data/shares')
```
Use `sharesDir` everywhere instead of the hardcoded path. This makes the endpoints work in `nuxt dev` too (set `DATA_DIR=./data` in `.env`).

**composables/useUrlState.ts**: Replace the existing implementation entirely.

The new `useUrlState` must:
1. Read `?s=token` from `route.query.s` (instead of `?v=compressed`)
2. Keep backward compat: also check `?v=` and decode it via existing `decodeViewState` for old links
3. Expose `copyShareLink(state, totalNodes)` that POSTs to /api/share and copies the resulting URL to clipboard
4. Expose `fetchSharedState()` that reads `?s=token` and GETs /api/share/{token}

New implementation:
```typescript
import { computed } from 'vue'
import { decodeViewState } from '@/utils/state-encoding'
import type { ViewState } from '@/utils/state-encoding'

export function useUrlState() {
  const route = useRoute()
  const router = useRouter()

  // Backward compat: decode old ?v= compressed links
  const legacyUrlState = computed<ViewState | null>(() => {
    if (!route.query.v) return null
    return decodeViewState(route.query as Record<string, string>)
  })

  // Check if current URL has a server-side share token
  const shareToken = computed(() => route.query.s as string | undefined)

  /**
   * Fetch ViewState from server-side share token (?s=token).
   * Returns null if no token or fetch fails.
   */
  const fetchSharedState = async (): Promise<ViewState | null> => {
    // Prefer new ?s= token
    if (shareToken.value) {
      try {
        const data = await $fetch<ViewState>(`/api/share/${shareToken.value}`)
        return data
      } catch {
        return null
      }
    }
    // Fall back to legacy ?v= compressed param
    return legacyUrlState.value
  }

  /**
   * POST state to /api/share, replace URL with ?s=token, copy to clipboard.
   * Returns true on success, false on failure.
   */
  const copyShareLink = async (state: ViewState, _totalNodes: number): Promise<boolean> => {
    try {
      const { token } = await $fetch<{ token: string }>('/api/share', {
        method: 'POST',
        body: state,
      })
      // Replace URL with short token — remove any old ?v= param
      await router.replace({ query: { s: token } })
      await nextTick()
      await navigator.clipboard.writeText(window.location.href)
      return true
    } catch {
      return false
    }
  }

  return { shareToken, fetchSharedState, copyShareLink }
}
```
