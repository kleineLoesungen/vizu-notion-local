# Quick Task 260604-v25: Filter Additional Sources in Metro Multi-Source Selector

**One-liner:** Added `eligibleAdditionalSources` computed that filters metro overlay candidates to only sources with a `date` columnMapping role, removing misleading non-timeline-positionable entries.

## What Was Done

Added a `eligibleAdditionalSources` computed ref in `pages/visualizations/[sourceId].vue` that replaces the raw `allSources` reference in the metro multi-source checkbox section.

**Computed logic:**
- Excludes the current primary source (`src.id !== sourceId.value`)
- Excludes sources without a `date` role in `columnMappings` (`'date' in (src.columnMappings ?? {})`)

**Template changes:**
- `v-if` guard changed from `allSources.length > 1` to `eligibleAdditionalSources.length > 0` — section hidden entirely when no eligible overlays exist
- `v-for` changed from `allSources` to `eligibleAdditionalSources`
- Removed `:disabled`, conditional `:class` on label and input — no longer needed since the primary source is excluded at the computed level rather than visually disabled
