## What Was Built

### Task 1: join-rows and lookup-by helpers (server/utils/templates.ts)

Two new Handlebars helpers registered after the existing `palette` helper at line 149:

**`join-rows`** (`(join-rows ArrayA "fieldA" ArrayB "fieldB" "prefix")`):
- Builds a Map index on arrayB keyed by `fieldB` values — O(n) lookup
- For each rowA, finds all matching rowBs where `rowB[fieldB] === rowA[fieldA]`
- Merges as `{ ...rowA, prefix_key: val }` for every matched rowB
- Left-join: rows with no B match included as `{ ...rowA }` unchanged
- Returns `[]` on invalid input (non-array args, missing field names)

**`lookup-by`** (`(lookup-by Array "field" value)`):
- Filters array to rows where `String(row[field]) === String(value)`
- Enables nested `{{#each}}` patterns to iterate cross-source matching rows
- Returns `[]` on invalid input (non-array, missing field)
